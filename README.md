# Book Review App

A simple Book Review application built using Node.js, Express.js, and EJS. It allows users to add, view, edit, and delete book reviews using RESTful routes.

## Features

- Add a new book review
- View complete review
- Delete a review
- RESTful routing

## Tech Stack

- Node.js
- Express.js
- EJS
- HTML
- CSS
- JavaScript
- Method Override
- UUID

## Installation

```bash
git clone https://github.com/meghal14/book-review-app.git

cd book-review-app

npm install

node index.js
```

Open your browser and visit:

```
http://localhost:4040
```
